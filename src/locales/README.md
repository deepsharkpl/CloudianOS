# Locales (i18n)

This project includes a `locales/` folder that contains all translations used in the application.

Each language is stored separately and follows the same structure to ensure consistency across the system.

## 🌍 Adding a New Language

If you want to add your own language, you can do it freely by following the existing structure.

Please:
- create a new folder or file based on the current pattern
- use the English version as the reference source
- translate the English content into your own language
- keep all keys, structure, and formatting identical

👉 The English version is the base reference and should be used as the main source for all translations.

## 📌 Guidelines

- Do not modify the structure of existing translation files
- Only translate values, not keys
- Keep naming conventions consistent
- Ensure completeness of translation (no missing fields)

---

This system allows the application to support multiple languages in a scalable and consistent way.
