# README

## /users Directory

Welcome to the `/users` directory.

This folder is intended for user applications, personal projects,
custom software, and user-created files.

All non-system applications should be stored here.

Example structure:

/users/my_application
/users/projects
/users/tools

The `/users` directory allows users to safely create, modify,
and manage their own applications without affecting the operating system.

Do not place system-critical files in this directory.
System applications belong in `/system`.
